Endpoints
GET
/api/risk-score
Single token risk score
Query Parameters
token string required CoinGecko ID, name, or symbol. PulseGuard finds the closest match automatically. e.g. solana, btc, bonk
Example Request
GET /api/risk-score?token=solana
Response
{
  "token": {
    "name": "Solana",
    "symbol": "sol",
    "image": "https://..."
  },
  "price": 68.14,
  "change_24h": 2.18,
  "risk_score": 43,
  "risk_level": "Medium",
  "risk_color": "#f5c542",
  "breakdown": {
    "volatility": 4,
    "liquidity": 20,
    "momentum": 18
  },
  "ai_insight": "Solana shows moderate risk..."
}

GET
/api/risk-score?tokens=...
Batch — up to 10 tokens
Query Parameters
tokens string required Comma-separated list of token IDs/names. Max 10. Batch responses use rule-based insights (no AI per token) for faster response times.
Example Request
GET /api/risk-score?tokens=solana,bitcoin,ethereum
Response
{
  "results": [
    { "token": { "name": "Solana" }, "risk_score": 43, "risk_level": "Medium", ... },
    { "token": { "name": "Bitcoin" }, "risk_score": 28, "risk_level": "Low", ... },
    { "token": { "name": "Ethereum" }, "risk_score": 31, "risk_level": "Low", ... }
  ]
}
